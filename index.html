研究背景

本项目复现 Mango-Mamba：一种基于 Mamba 架构的芒果叶病害轻量级检测模型。该模型采用三阶段课程学习策略 (Three-Phase Curriculum Training)，在 VN-MangoLeaf 数据集上实现 7 类芒果叶病害的分类检测。

复现环境要求： Google Colab + T4 GPU

数据集

VN-MangoLeaf 数据集包含 7 类芒果叶病害图像：

类别 描述

Powdery_Mildew 白粉病
Red_Rust 红锈病

模型训练

训练代码采用三阶段课程学习策略，核心超参数如下：

  • Batch Size: 32
  • Epochs: 30
  • Learning Rate: 1e-3 (AdamW + CosineAnnealing)
  • 数据划分: 80% 训练 / 20% 验证

Grad-CAM 热力图

使用 Grad-CAM 可视化模型关注区域，帮助理解模型的决策依据：

import cv2

class GradCAM:
    def __init__(self, model, target_layer):
        self.model = model
        self.gradients = None
        self.activations = None
        target_layer.register_forward_hook(self.save_activation)
        target_layer.register_full_backward_hook(self.save_gradient)

    def save_activation(self, module, inp, out):
        self.activations = out

    def save_gradient(self, module, grad_in, grad_out):
        self.gradients = grad_out[0]

    def __call__(self, x):
        self.model.zero_grad()
        out = self.model(x)['logits']
        class_idx = out.argmax(1).item()
        out[0, class_idx].backward()
        weights = torch.mean(self.gradients, dim=[2,3], keepdim=True)
        cam = torch.sum(weights * self.activations, dim=1).squeeze()
        cam = torch.relu(cam)
        cam = cam - cam.min()
        cam = cam / cam.max()
        return cam.cpu().detach().numpy(), class_idx

# 为每个类别生成 Grad-CAM
grad_cam = GradCAM(model, model.norm)
for cls_name in CLASS_NAMES:
